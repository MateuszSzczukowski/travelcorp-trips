# Recruitment task for Travcorp

The project was made for recruitment into the Travcorp company.

## Technologies

| Frontend  | Backend  |
| --- | --- |
| - React | - Node |
| - SCSS | - Express |
## Instruction 

If you want to run the project follow the instructions below.

| Frontend  | Backend  |
| --- | --- |
| cd frontend | cd backend |
| npm i | npm i |
| npm start | npm start |
